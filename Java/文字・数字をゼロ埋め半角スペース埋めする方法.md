# 文字・数字をゼロ埋め半角スペース埋めする方法

```java
//文字の先頭を8桁で空白埋めする
String.format("%8s", 123456);

//文字の後ろを8桁で空白埋めする
String.format("%-8s", 123456);

//文字の先頭を0埋めする
String.format("%08s", 123456);


//おまけ
//数字を3桁毎にカンマ区切りする
String.format("%,d", 123456789)
```

### 出力結果
```
[  123456]  //先頭
[123456  ]  //後ろ
[00123456]  //先頭0埋め
[123,456,789]  //3桁毎にカンマ区切り
```
